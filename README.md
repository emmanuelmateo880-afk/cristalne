# CRISTALNE - Premium Streaming Platform

A production-ready Netflix-style streaming platform built with Next.js 15, React, TypeScript, and PostgreSQL.

## Features

- 🎬 Movies, TV Series, Animations, and Cartoons
- 👤 User authentication and profiles
- ❤️ Favorites and watch history
- 💳 Subscription management (Stripe & PayPal)
- 📺 Video streaming with iframe support
- 🎯 Advanced search and filtering
- 📊 Admin dashboard
- 💰 Google AdSense integration
- 🚀 Production-ready deployment

## Tech Stack

- **Frontend**: Next.js 15, React 19, TypeScript
- **Styling**: Tailwind CSS
- **Database**: PostgreSQL with Prisma ORM
- **Authentication**: NextAuth.js
- **Payment**: Stripe & PayPal
- **Deployment**: Docker
